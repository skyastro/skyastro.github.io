Release 2.0.0: July 28, 2023
  • Facilities for calculating and creating trajectories have been added to the program. Users can either manually control a rocket or write a program to do the control which can include atomic events, like specified thrusts or impulses, or calculated trajectories which the program tries to find the 'best' set of impulses for a given transfer. See the Trajectory program documentation for more information.
  • Earth orbiting articial satellites are only shown after the year in which the associated mission was launched.

Release 1.0.0: November 25, 2021
The initial release. This version includes:
  • Animation of the solar system including moons, small bodies and interplanetary missions
  • Animation of Earth system include the Moon and NORAD tracked bodies
  • Animation of the Local Galactic Neighborhood based on GAIA DR3 data (with fallback to XHip catalog)
  • Ability to derive unknown orbital parameters from known data.
Orbits are specified at one epoch and not interpolated other than for precession of moon and satellite orbits. Capabilities for visualizing the neighborhood or our Galaxy and for tracking accelerated trajectories are not included.
